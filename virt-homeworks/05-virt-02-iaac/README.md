## Задача 1

- Опишите своими словами основные преимущества применения на практике IaaC паттернов.
- Какой из принципов IaaC является основополагающим?
- 
### Ответ
1. Значительно ускоряет процесс предоставленния инфраструктуры, разработки, непрерывная сборка и интеграция, стабильность.
2. Идемпотентность - при выполнении одной и той же операции получаем одинаковый результат

## Задача 2

- Чем Ansible выгодно отличается от других систем управление конфигурациями?
- Какой, на ваш взгляд, метод работы систем конфигурации более надёжный push или pull?

### Ответ
1. Использует существующую SSH инфраструктуру.
2. Pull кажется более надежной из-за того что сервер сам идет за своей конфигурацией, нет зависимости от доступности сервера.
